# Roadmap

## MVP (6-8 weeks)
- Local bookmarks, tags, and search.
- Workspace and tab model in UI.
- Rule-based auto-tagging.
- Export/import of local data.

## v1.0 (12-16 weeks)
- LLM tagging worker integration.
- Encrypted multi-device sync.
- macOS packaging and distribution.
- Performance tuning and privacy controls.
